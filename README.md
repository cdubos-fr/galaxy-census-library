# Installation

```bash
$ pip install <path-to-this-project>
```
or
```bash
$ pip install git+<git-url>
```

to install it in editable mode:
```bash
$ pip install -e .
```

to add developpement dependencies:
```bash
$ pip install -e ".[dev]"
```

and use pre-commit to check your code
```bash
$ pre-commit install
```
